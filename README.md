# KLYX - Política de Privacidad

Este repositorio contiene la política de privacidad oficial del proyecto KLIX.

## 🌐 Publicación

La política está publicada mediante **GitHub Pages** y es accesible en:

https://klyx-lab.github.io/privacy-policy-klyx/

## 📌 Uso

Este documento se utiliza como referencia legal para:

- Bots de Telegram
- Servicios digitales
- Automatización de procesos
- Atención al cliente

## ⚖️ Contenido

Incluye información sobre:

- Responsable del tratamiento
- Finalidad del uso de datos
- Tipos de datos recogidos
- Base legal (RGPD)
- Conservación de datos
- Derechos del usuario
- Información sobre servicios del bot

## 🚀 Notas

- Repositorio público
- Página estática sin backend
- Compatible con integración en Telegram (BotFather)
